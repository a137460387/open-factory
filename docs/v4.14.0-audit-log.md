## 点 2 — CI Locale 实测数据

### 实测方法

在 `ShortcutCheatsheetPanel.test.tsx` 的 `beforeAll` 中加入 debug `console.log`，推送到 `chore/debug-ci-locale` 分支，从 GitHub Actions 日志提取真实值。

### 实测数据

| 指标 | 值 |
|------|----|
| `typeof navigator` | `object` |
| `navigator.language` | `en-US` |
| `getLanguage()` (setLanguage 前) | `en` |
| `getLanguage()` (setLanguage("zh") 后) | `zh` |
| 渲染 HTML 含 | `"Shortcut Cheatsheet"` (英文) |

### CI 运行记录

| 运行 | 分支 | frontend job | e2e job |
|------|------|-------------|---------|
| [28609548344](https://github.com/a137460387/open-factory/actions/runs/28609548344) | main (147e9417) | ❌ failure (locale mismatch) | ⏳ in_progress |
| [28613255158](https://github.com/a137460387/open-factory/actions/runs/28613255158) | chore/debug-ci-locale | ✅ success | ⏳ in_progress |

### 结论

**测试 bug，非产品 bug**。CI runner `navigator.language=en-US` → `normalizeLanguage()` → `currentLanguage=en` → 渲染英文字符串。
修复方案：测试 `beforeAll` 中加 `setLanguage('zh')`。
