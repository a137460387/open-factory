# Open Factory v4.44.0 Release Notes

**Sprint S — 自动剪辑生成器与模板化创作系统**

发布日期：2026-07-19

---

## 🤖 自动剪辑生成器

基于场景分析和模板规则自动生成时间线，支持 BPM 卡点对齐。

- 场景检测驱动的智能剪辑点生成
- BPM 节拍同步卡点，自动匹配音乐节奏
- 支持自定义生成参数：片段时长范围、转场偏好、特效密度
- 与现有时间线系统无缝集成，生成结果可手动微调
- Worker 线程异步执行，UI 全程零阻塞

## 📐 模板化创作系统

内置预设模板库，支持模板 CRUD 和导入导出。

- 预置多种创作风格模板（Vlog、旅拍、产品展示、教程等）
- 模板包含完整的轨道结构、转场配置和效果预设
- 支持从现有项目保存为模板
- 模板导入/导出（JSON 格式），便于团队共享
- 模板参数化：可配置的占位符和变量

## 🧠 个人风格记忆库

学习用户剪辑习惯，持续优化生成权重。

- 记录用户对生成结果的修改和偏好
- 基于历史数据构建个人风格模型
- 自动生成权重调整建议
- 风格记忆数据本地存储，隐私安全
- 支持重置和导出风格数据

## ⚡ Worker 线程集成

生成算法在 Worker 线程执行，确保 UI 零阻塞。

- `automation.worker.ts` 封装所有计算密集型任务
- `useAutomationWorker` Hook 提供类型安全的通信接口
- 支持任务取消和进度回调
- 错误边界隔离，Worker 异常不影响主 UI
