# Open Factory v4.35.0 — AI 视觉分析与智能剪辑、插件系统架构

> 🎉 **里程碑：智能创作平台与开放生态启航**

## 🎬 AI 视觉分析

基于 CLIP 模型的视觉理解能力，为视频编辑带来智能化场景分析。

| 功能 | 描述 |
|------|------|
| **场景自动检测** | 基于 CLIP 模型的视觉特征提取，自动识别视频中的场景切换点，支持阈值可调的灵敏度配置 |
| **相似场景搜索** | 通过视觉特征向量比对，在项目中快速查找相似画面，适用于素材整理与内容复用 |
| **场景分类标注** | 自动为检测到的场景生成语义标签（室内/室外、人物/风景等），便于素材管理 |

**核心文件**: `packages/editor-core/src/ai/scene-detection.ts`

## ✂️ 智能剪切建议

结合音频 VAD（语音活动检测）与视觉分析，智能标记冗余片段，提升剪辑效率。

| 剪切类型 | 检测方式 |
|----------|----------|
| **静音片段** | 基于 VAD 的语音活动检测，自动标记无对话的静音区间 |
| **静态帧** | 视觉特征变化率低于阈值时，识别为静态/无效画面 |
| **低能量片段** | 音频能量分析，标记音量过低的无效区间 |
| **重复内容** | 结合视觉与音频特征，识别重复拍摄的素材 |

**核心文件**: `packages/editor-core/src/ai/smart-cut.ts`

## 🖼️ 自动重构（Auto Reframe）

主体追踪与动态裁剪技术，一键适配多平台宽高比。

| 平台 | 宽高比 | 说明 |
|------|--------|------|
| YouTube / 横屏 | 16:9 | 标准横屏格式 |
| Instagram / 正方形 | 1:1 | 正方形裁剪 |
| TikTok / 竖屏 | 9:16 | 竖屏短视频格式 |
| Instagram Stories | 9:16 | 全屏竖屏故事格式 |
| 电影宽银幕 | 21:9 | 超宽银幕格式 |

- **主体追踪**: 自动识别画面中的主要运动主体，确保裁剪后主体始终在画面中心
- **动态裁剪**: 平滑的裁剪窗口移动，避免画面跳动
- **批量处理**: 支持对整个项目或选定片段批量应用重构

**核心文件**: `packages/editor-core/src/ai/auto-reframe.ts`

## 🧩 插件系统架构

建立完整的四类插件标准接口，实现生命周期管理与权限系统，为开放生态奠定基础。

### 插件类型

| 类型 | 用途 | 示例 |
|------|------|------|
| **Effect Plugin** | 自定义视频/音频效果 | LUT 滤镜、粒子特效、转场效果 |
| **Export Plugin** | 自定义导出格式与目标 | 平台直传、云存储、编码器扩展 |
| **Workflow Plugin** | 自动化工作流 | 批量处理、模板应用、项目管理 |
| **AI Model Plugin** | 集成第三方 AI 模型 | 语音识别、图像生成、风格迁移 |

### 架构特性

- **生命周期管理**: 插件的加载、初始化、激活、停用、卸载全流程管理
- **权限系统**: 细粒度的权限控制（文件访问、网络请求、UI 修改等）
- **沙箱隔离**: 插件运行在独立沙箱中，确保核心系统稳定
- **版本兼容**: 插件 SDK 版本管理，确保向后兼容
- **热加载**: 开发模式下支持插件热加载，提升开发效率

**核心文件**:
- `packages/editor-core/src/plugins/plugin-registry.ts`
- `packages/editor-core/src/plugins/plugin-types.ts`
- `packages/plugin-sdk/` (SDK 包)

## 🏗️ 架构遵循

本版本严格遵循项目架构规范：

- ✅ 核心算法在 `packages/editor-core`，保持 80%+ 覆盖率
- ✅ Timeline 突变通过 Command 对象，不直接调用 Zustand setter
- ✅ 所有 Tauri 调用通过 `tauri-bridge.ts` 统一封装
- ✅ 插件 SDK 独立包，与核心解耦
- ✅ 无第三方代码复制，保持 clean-room 实现
- ✅ 无遥测/登录/云服务，保持 local-first 原则
