# Open Factory v4.66.0 Release Notes

**发布日期**: 2026-07-23

## 核心特性

### 🧠 预测性 AI 引擎

- **视觉高光检测** (`visual-highlight-engine.ts`) — 自动分析视频帧，识别高对比度、运动模糊、人脸聚集等视觉亮点，生成时间轴标记
- **音频节奏分析** (`audio-rhythm-analysis.ts`) — 基于 onset detection 和 BPM 估计算法，从音频波形中提取节拍点，支持剪辑点自动吸附
- **智能粗剪方案生成** (`smart-rough-cut.ts`) — 综合视觉高光、音频节奏、场景变化等多维信号，自动生成多种粗剪方案供创作者选择

### 🗣️ 多模态交互

- **全局命令面板** (`CommandPalette.tsx`) — `⌘+K` 一键唤起，支持自然语言指令输入，直接驱动编辑操作
- **自然语言指令引擎** (`natural-language-commands.ts`) — 解析中文/英文指令（如"把第二段视频加速2倍"、"在10秒处添加转场"），映射为结构化编辑命令
- **手势控制集成** (`gesture-control.ts` + `GestureTutorial.tsx`) — 支持滑动、缩放、旋转等触控手势，配合交互式教程引导用户上手

### 🎯 上下文感知

- **时间线动态建议气泡** (`ContextualSuggestionBubble.tsx`) — 根据当前播放位置、片段类型、编辑历史，实时推荐转场效果、调色预设、音频处理
- **上下文建议引擎** (`contextual-suggestions.ts`) — 分析时间线上下文（相邻片段、时间窗口、媒体类型），生成个性化创作建议

### 🎵 节拍吸附

- **精细波形显示** (`AudioWaveformDisplay.tsx`) — 时间线内嵌高分辨率音频波形，节拍标记可视化
- **剪辑点自动对齐** — 吸附模式下，剪辑点自动对齐最近的音频节拍，提升音画同步精度
