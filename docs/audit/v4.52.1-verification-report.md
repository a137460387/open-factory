## 一、P0 修复验证

### 1.1 TaskScheduler 时间片超时逻辑 (HIGH-2) ✅

**文件**: `packages/editor-core/src/core/task-scheduler.ts`

**修复内容**:
- 添加 `TimeSliceExceeded` 错误类型区分（`error.name = 'TimeSliceExceeded'`）
- `executeTask` 中识别时间片超时，不计入重试次数
- 添加 `isTimeSliceExceeded()` 辅助方法

**验证**: 时间片超时的任务被正确重新入队，不触发 retry 计数。

### 1.2 帧缓存 MinHeap 优化 (HIGH-1, MEDIUM-3) ✅

**文件**: `packages/editor-core/src/render-cache.ts`, `apps/desktop/src/lib/preview/gpu-acceleration.ts`

**修复内容**:
- 为 `TimelineRenderFrameCache` 和 `GpuTexturePool` 添加 MinHeap 数据结构
- 添加 `reheapify()` 方法，在 `get()` 更新优先级后恢复堆序
- `pruneToBudget()` 使用懒删除跳过陈旧堆条目
- 淘汰复杂度从 O(n log n) 降至 O(log n)

**验证**: 所有 17 个缓存相关测试通过，LRU 淘汰顺序正确。

### 1.3 任务饥饿预防 (HIGH-3) ✅

**文件**: `packages/editor-core/src/core/task-scheduler.ts`

**修复内容**:
- 添加 `boostPriorityForPreemptedTask()` 方法
- 被抢占 3 次后临时提升优先级至 `normal`
- 保留原始优先级信息以支持恢复

**验证**: 低优先级任务在多次被抢占后获得提升，不会永久饥饿。
