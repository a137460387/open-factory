# Open Factory v4.56.0 Release Notes

**发布日期**: 2026-07-22
**Sprint**: AF - 服务化与生态扩展

---

## 概述

v4.56.0 标志着 Open Factory 从本地桌面编辑器正式升级为**创作平台**。本次更新通过 5 个并行开发轨道，交付了完整的协作服务化架构和开发者生态系统，让引擎真正运转起来，连接外部世界。

---

## 新增功能

### 📡 协作中继服务

独立的 WebRTC 信令/中继服务器，支持多人实时协作编辑。

- **WebRTC 信令**: offer/answer 交换、ICE 候选处理、TURN 服务器配置
- **房间管理**: 创建/加入/离开房间、房间状态机（waiting → active → closed）
- **协作者管理**: 用户列表、光标位置广播、编辑操作同步
- **权限控制**: owner/editor/viewer 三级权限体系
- **Redis 集群**: 房间状态持久化、跨实例 pub/sub 同步
- **管理 API**: REST 端点管理房间和用户
- **Docker 部署**: 多阶段构建、非 root 用户、健康检查、资源限制

**包路径**: `packages/collaboration-server/`

### 🛒 插件市场 2.0

扩展插件规范，支持更丰富的插件生态。

- **沙箱隔离**: 基于权限配置的自动隔离环境，支持同步/异步执行、超时控制、违规追踪
- **权限控制**: 声明式权限（filesystem/network/process/ui），全局拒绝列表，完整审计日志
- **CLI 命令注册**: 插件可注册自定义 CLI 命令，支持参数解析和帮助生成
- **工作流节点**: 插件可定义工作流节点，支持分类目录和连接验证
- **搜索引擎**: 多字段加权评分（name 100分、keywords 60分、description 40分等）
- **版本管理**: semver 解析/比较/范围匹配，兼容性检查，自动更新检测

**包路径**: `packages/plugin-market/`

### 🧩 TypeScript SDK

完整的 TypeScript SDK，封装核心 API 供外部开发者使用。

- **OpenFactoryClient**: 统一客户端类，链式 API 设计
- **ProjectAPI**: 项目创建、打开、保存、配置管理
- **TimelineAPI**: 时间线操作，添加/删除/移动片段
- **EffectsAPI**: 特效管理，应用/更新/移除特效
- **ExportAPI**: 导出功能，进度追踪
- **PluginsAPI**: 插件管理，注册/启用/禁用
- **EventEmitter**: 事件驱动架构，支持状态变更订阅
- **React 组件库**: `OpenFactoryProvider`, `useProject`, `useTimeline`, `useEffects`, `useExportProgress`
- **Vue 组件库**: `OpenFactoryPlugin`, `useProject`, `useTimeline`, `useEffects`, `useExportProgress`

**包路径**: `packages/sdk/`

### 📚 开发者文档

基于 Docusaurus 3.x 构建的完整开发者文档站点。

- **快速开始指南**: 环境配置、项目结构、包概览
- **API 参考**: Editor Core、Plugin SDK、CLI、TypeScript SDK 完整文档
- **开发指南**: 插件开发、协作服务部署、工作流节点开发
- **贡献指南**: 开发流程、编码规范、PR 流程

**文档路径**: `docs/developer/`

### 📈 创作者激励计划

完整的创作者后台和激励体系。

- **数据分析仪表盘**: 下载量、活跃用户、评分趋势、收入明细
- **阶梯分成机制**:
  - 入门级: 70% 创作者 / 30% 平台
  - 进阶级 (累计 ¥1,001+): 75% / 25%
  - 专业级 (累计 ¥10,001+): 80% / 20%
  - 旗舰级 (累计 ¥50,001+): 85% / 15%
- **创作者等级体系**: 自动升级、专属权益
- **收入管理**: 分成计算、提现、账单、税务

**包路径**: `packages/creator-dashboard/`
**文档路径**: `docs/creator-program/`

## 升级指南

### 新增依赖

本次更新新增 4 个内部包，无需修改现有代码。如需使用新功能：

```typescript
// 使用 TypeScript SDK
import { OpenFactoryClient } from '@open-factory/sdk';

const client = new OpenFactoryClient();
client.project.create({ name: 'My Video', width: 1920, height: 1080, fps: 30 });
```

### 协作服务器部署

```bash
# Docker 部署
cd packages/collaboration-server
docker-compose up -d
```
