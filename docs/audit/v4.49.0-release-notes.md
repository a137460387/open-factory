# Open Factory v4.49.0 Release Notes

**Sprint X: 极致本地化与 LLM 语义协同引擎**

发布日期: 2026-07-20

---

## 核心架构突破

### "本地特征提取 + 云端逻辑推演" 双层架构

v4.49.0 确立了个人本地 AI 自动化的核心架构范式：

- **本地层**: 视频关键帧抽取、ASR 语音识别、场景识别等重计算任务在用户设备完成
- **云端层**: 远端 LLM 基于结构化元数据生成剪辑指令，原始素材绝不上传
- **隐私保障**: 用户的视频、音频等原始数据始终留在本地，仅传输语义特征

---

## 新功能模块

### 1. 本地语义提取引擎 (`semantic-extractor.ts`)

从视频素材中提取结构化语义元数据：

- **关键帧抽取**: 智能识别场景切换点，提取代表性帧
- **ASR 语音识别**: 本地化语音转文字，生成带时间戳的字幕数据
- **场景识别**: 分析画面内容，识别场景类型、人物、物体
- **元数据输出**: 生成标准化 JSON 结构，供 LLM 消费

### 2. LLM 语义协同引擎 (`llm-orchestrator.ts`)

基于本地提取的元数据，调用远端大模型生成剪辑指令：

- **智能编排**: 根据素材语义自动规划剪辑流程
- **指令生成**: 输出结构化剪辑指令（裁剪点、转场、特效等）
- **上下文管理**: 维护多轮对话上下文，支持迭代优化
- **多模型适配**: 支持 OpenAI、Claude、本地 Ollama 等多种 LLM

### 3. 多平台智能发布引擎 (`platform-publisher.ts`)

对接主流视频平台，实现一键多平台发布：

- **平台适配**: 内置抖音、B站、YouTube、小红书等平台规范
- **智能生成**: LLM 自动生成标题、标签、描述
- **批量发布**: 支持同时发布到多个平台
- **格式转换**: 自动适配各平台的分辨率、码率要求

## UI 面板

### 1. AI 索材分析面板 (`semantic-panel.ts`)

- 展示本地语义提取结果
- 可视化关键帧、场景分割、语音波形
- 支持手动修正提取结果

### 2. AI 对话剪辑面板 (`dialogue-panel.ts`)

- 自然语言交互式剪辑
- 实时预览 LLM 生成的剪辑方案
- 支持多轮对话迭代优化

### 3. 多平台发布面板 (`publish-panel.ts`)

- 多平台账号管理
- 发布任务队列
- 发布状态实时追踪
