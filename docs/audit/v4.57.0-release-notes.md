## 🌐 插件市场 Web UI

Next.js 前端应用，为 Open Factory 插件生态系统提供完整的浏览、搜索和管理体验。

### 核心组件 (13 个)

| 组件 | 功能 |
|------|------|
| **SearchBar** | 全文搜索插件，支持关键词自动补全 |
| **CategoryNav** | 分类导航，支持多级分类筛选 |
| **PluginCard** | 插件卡片展示，包含评分、下载量、作者信息 |
| **PluginGrid** | 响应式网格布局，自适应多端展示 |
| **FeaturedCarousel** | 精选插件轮播推荐 |
| **TrendingList** | 热门趋势排行榜 |
| **InstallButton** | 一键安装/卸载交互 |
| **PermissionDialog** | 插件权限确认弹窗 |
| **RatingStars** | 五星评分组件 |
| **VersionHistory** | 版本历史记录展示 |
| **ReviewList** | 用户评论列表 |
| **ScreenshotGallery** | 截图画廊浏览 |
| **PluginDetail** | 插件详情页（完整信息展示） |

### 技术栈

- Next.js 14 + React 18
- TypeScript 全量类型覆盖
- 响应式设计，支持桌面端和移动端

## 📊 创作者后台 Web UI

React 数据仪表盘，为插件创作者提供收入分析、用户洞察和社区管理能力。

### 核心组件 (15 个)

| 组件 | 功能 |
|------|------|
| **KPICard** | 关键指标卡片（下载量、收入、评分） |
| **TrendChart** | 趋势折线图，可视化关键指标变化 |
| **SourcePieChart** | 来源分布饼图，分析流量来源 |
| **RatingBarChart** | 评分分布柱状图 |
| **RevenueLineChart** | 收入曲线图，追踪收入变化 |
| **TransactionTable** | 交易明细表格 |
| **WithdrawalForm** | 提现申请表单 |
| **WithdrawalHistory** | 提现历史记录 |
| **CreatorProfile** | 创作者个人档案 |
| **TierBadge** | 等级徽章体系 |
| **PluginList** | 我的插件管理列表 |
| **AchievementCard** | 成就系统卡片 |
| **NotificationList** | 通知消息列表 |
| **Layout** | 页面布局框架 |
| **Sidebar** | 侧边导航栏 |

### 技术栈

- React 18 + Vite
- Recharts 数据可视化
- TypeScript 全量类型覆盖
