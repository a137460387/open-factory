## Track B: 本地生成式 AI

### 新增包

| 包 | 用途 | 关键技术 |
|----|------|----------|
| `packages/ai-generator` | 视频生成 | Wan2.1 + WebGPU + 量化 |

### 支持模型

| 模型 | 参数 | 用途 | 精度 |
|------|------|------|------|
| Wan2.1-I2V-14B-720P-Turbo | 14B | 图片→视频 | FP16/INT8/INT4 |
| Wan2.1-I2V-1.3B-480P | 1.3B | 轻量图片→视频 | FP16/INT8/INT4 |
| Wan2.1-T2V-14B-720P | 14B | 文本→视频 | FP16/INT8/INT4 |

### 计算引擎

```
WebGPU (首选) → WebGL (降级) → CPU (兜底)
```

- 运行时自动检测 GPU 能力
- 无缝降级，无需用户干预
- LRU 模型缓存，支持热加载/卸载

### 生成管线

- **文本→视频**: prompt → 文本编码 → 帧序列 → 视频编码
- **图片→视频**: 图片 → 运动估计 → 帧序列 → 视频编码
- 运动强度可调，分辨率 720P/480P，帧率 16/24/30 fps

## 架构决策

1. **Fastify over Express** — 更好的 TypeScript 支持和性能
2. **单例 API 客户端** — React 组件共享同一实例
3. **结构化类型对齐** — 三端类型一致，避免运行时转换
4. **WebGPU 自动降级** — 最大化硬件兼容性
5. **INT8 默认精度** — 14B 模型最佳质量/体积平衡
6. **Zod 配置校验** — 类型安全的配置管理
