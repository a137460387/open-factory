# Open Factory v4.67.0 Release Notes

**Sprint AU — 极限性能调优与内存重构**

> 底层极限重构，在重度 AI 功能下捍卫无感流畅体验。

---

## ⚡ 渲染性能突破

**问题**：EditorShell 巨型组件在时间线拖拽时触发 60fps 全量重渲染，帧率骤降至 15fps 以下。

**方案**：阻断高频重渲染路径，将 EditorShell 拆分为独立渲染区域，仅在数据实际变化时触发局部更新。时间线拖拽操作帧率恢复至 60fps 丝滑流畅。

- `apps/desktop/src/components/EditorShell.tsx` — 拆分渲染边界，阻断级联重渲染

---

## 🧵 多线程波形渲染

**问题**：AudioWaveformDisplay 在主线程绘制波形，阻塞 UI 交互。

**方案**：将波形绘制迁移至 OffscreenCanvas + Worker 线程，主线程零阻塞。

- `apps/desktop/src/components/Timeline/AudioWaveformDisplay.tsx` — 迁移至 OffscreenCanvas
- `apps/desktop/src/workers/waveform-render.worker.ts` — 新增 Worker 波形渲染器

---

## 🧠 AI 智能调度

**问题**：后台 AI 推理任务（场景检测、智能剪辑等）抢占主线程资源，影响用户交互。

**方案**：引入 PriorityScheduler，用户交互时自动挂起后台 AI 推理，操作绝对流畅。

- `apps/desktop/src/engine/priority-scheduler.ts` — 优先级任务调度器
- `apps/desktop/src/engine/model-manager.ts` — AI 模型按需加载与自动卸载

---

## 💾 内存优化重构

**问题**：GPU 显存碎片化，AI 模型重复加载导致内存峰值过高。

**方案**：GPU 显存池化复用，AI 模型按需加载与自动卸载。

- `apps/desktop/src-tauri/src/zero_copy_io.rs` — Rust 侧零拷贝 IO
- `apps/desktop/src/engine/gpu-memory-pool.ts` — GPU 显存池化
- `apps/desktop/src/lib/tauri-bridge/ipc-optimizer.ts` — IPC 通信优化
