## 🎉 本次更新亮点

v4.30.0 为 Open Factory 带来了专业级的音频处理能力，包括多轨混音器、实时频谱分析和智能降噪三大核心功能，全面提升了音频编辑的工作流程。

---

## 🎚️ 多轨混音器

全新的多轨混音器面板，提供 DAW 级别的音频混合体验：

### 通道条 (Channel Strip)
- **音量推子**: -60 dB ~ +12 dB 精确控制，步进 0.1 dB
- **声像旋钮**: -100 (L) ~ +100 (R) 立体声声像调节
- **静音/独奏**: 快速隔离和对比音轨
- **VU 电平表**: 实时电平监控，三色指示（绿/黄/红）
- **声道通道路由**: 支持立体声/单声道/路由/混合模式

### 效果架 (Effects Rack)
- **20 种内置效果**: EQ 4/8 段、压缩器、限制器、噪声门、扩展器、混响、延迟、合唱、镶边、失真、齿音消除、降噪、变调、立体声增强、M/S 处理、增益、相位反转、高通/低通滤波
- **效果链管理**: 支持添加、删除、排序、启用/禁用
- **Wet/Dry 混合**: 每个效果独立的干湿比控制（0-100%）

### 总线系统
- **多总线路由**: 灵活的音频信号路由
- **Master 输出**: 主音量控制和 VU 表监控
- **Ducking 自动闪避**: 支持主导/背景轨道选择，阈值/比率/起始/释放参数调节

### 自动化
- **自动化曲线编辑器**: 基于 SVG 的可视化编辑
- **4 种曲线类型**: 线性(linear)、贝塞尔(bezier)、阶梯(step)、平滑(smooth)
- **录制模式**: Read/Write/Touch/Latch 四种自动化录制模式

## 📊 实时音频可视化

### 频谱分析器 (Spectrum Analyzer)
- **三种显示模式**: 柱状图(bars)、线条(line)、填充(filled)
- **对数频率刻度**: 20 Hz ~ 20 kHz 全频段覆盖
- **dB 振幅刻度**: -60 dB ~ 0 dB 动态范围
- **峰值保持**: 带衰减动画的峰值指示
- **渐变颜色编码**: 绿色（正常）→ 黄色（高电平）→ 红色（削波）
- **HiDPI 支持**: 自适应高分辨率显示

### 频道分析面板
- **实时频率响应曲线**: SVG 渲染的频率响应可视化
- **相位示波器 (Phase Scope)**: 立体声相位关系监控
- **峰值频率显示**: 实时频率峰值检测
- **录制历史回放**: 最长 30 秒/60 帧的历史数据
- **JSON 导出**: 分析数据可导出为 JSON 格式

## 🎙️ 智能降噪

基于 FFmpeg `afftdn` 滤镜的一键智能降噪功能：

### 预设系统
| 预设 | 噪声底限 | 降噪类型 | 说明 |
|------|----------|----------|------|
| 轻度降噪 | -30 dB | 弱 (nr=0) | 适合轻微背景噪音 |
| 中度降噪 | -40 dB | 中 (nr=1) | 适合一般噪音环境 |
| 强力降噪 | -50 dB | 强 (nr=2) | 适合严重噪音环境 |
| 自定义 | 用户定义 | 用户定义 | 完全自定义参数 |

### 高级参数
- **噪声底限 (Noise Floor)**: -60 dB ~ 0 dB 精确控制
- **降噪类型**: 弱/中/强三档降噪算法
- **自动噪声采样**: 强度 >70% 时自动启用
- **噪声采样窗口**: 可指定采样起始/结束时间

### 效果预览
- **SNR 改善预估**: 实时计算信噪比改善值
- **处理前后峰值**: 显示降噪前后的峰值变化
- **FFmpeg 滤镜预览**: 实时显示生成的滤镜参数

### 导出集成
- **效果槽集成**: 降噪参数可转换为标准效果槽格式
- **导出管线支持**: 自动注入导出 FFmpeg 命令
- **按轨道配置**: 每个音轨可独立配置降噪参数
