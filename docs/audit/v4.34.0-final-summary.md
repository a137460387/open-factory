### 技术架构亮点

1. **本地 Whisper.cpp 集成**
   - 通过 WebAssembly 在桌面端运行 whisper.cpp 推理
   - 支持 tiny/base/small/medium 四种模型尺寸
   - 多语言支持：中/英/日/韩 + 自动检测

2. **非阻塞 UI 设计**
   - Web Worker 完全隔离推理计算
   - 三阶段进度回调：模型加载 → 音频预处理 → 推理转录
   - 主线程始终保持 60fps

3. **智能后处理流水线**
   - 短句合并、长句拆分
   - 时间戳精确对齐
   - 阅读时间自动估算

4. **字幕工作流深度集成**
   - 一键生成字幕并导入时间线
   - 无缝衔接 v4.31.0 字幕编辑器
   - 多格式导出兼容

## "本地优先 AI 架构"落地意义

v4.34.0 建立的本地 AI 架构模式将成为后续所有 AI 功能的基础：

1. **隐私安全范式**: 所有 AI 推理在用户设备完成，音频/视频数据永不离开本地
2. **离线可用**: 无需网络连接即可使用全部 AI 功能
3. **无供应商锁定**: 基于开源 whisper.cpp，不依赖任何云服务商
4. **可扩展架构**: Web Worker + WASM 模式可复用于未来的 AI 功能（场景检测、智能剪辑等）
