## 🎬 Sprint A — 智能粗剪面板 (Smart Rough Cut Panel)

全新智能粗剪面板，整合 7 类 AI 分析结果，提供一键式粗剪工作流。

### 核心特性

| 特性 | 描述 |
|------|------|
| **7 类 AI 分析串联** | 场景检测 → 静音检测 → 对话检测 → Whisper 字幕 → 节拍分析 → 情感分析 → 叙事结构，自动编排执行 |
| **一键粗剪工作流** | 统一编排面板（`SmartRoughCutOrchestratorPanel`），一键运行全部分析，自动生成剪辑建议 |
| **交互式建议调整** | 8 种建议类型（场景分割、静音移除、字幕添加、对话提取、B-roll 插入、节奏剪辑、情感高亮、叙事结构），支持拖拽排序、按类型筛选、批量选择 |
| **剪辑报告** | 结构化报告展示素材分析概览（场景边界数、静音段数、字幕条数等）、决策统计、预估输出时长 |
| **双面板设计** | 保留原始逐项面板（`SmartRoughCutPanel`）与新一代统一编排面板，按需切换 |

## ⚡ Sprint B — 导出性能升级 (Export Performance)

智能导出调度、实时帧预览、VMAF 质量监控三大能力全面升级。

### 核心特性

| 特性 | 描述 |
|------|------|
| **智能导出调度** | 根据项目复杂度自动选择最优编码参数（preset/CRF/线程数/硬件加速），支持速度/质量/平衡三种目标 |
| **实时帧预览** | 导出过程中通过 Canvas 实时显示已渲染帧缩略图，替代传统纯进度条 |
| **VMAF 质量监控** | 三级降级策略（实时 → 导出后 → 仅估算），自动采样并生成质量报告 |
| **队列集成** | 导出队列运行器自动集成智能调度，入队时自动应用最优参数 |

### 新增模块

| 模块 | 路径 | 行数 | 说明 |
|------|------|------|------|
| 导出调度器 | `packages/editor-core/src/export/export-scheduler.ts` | 605 | 项目复杂度分析、最优参数计算、调度决策 |
| VMAF 监控 | `packages/editor-core/src/export/vmaf-monitoring.ts` | 471 | 能力检测、采样计划、结果分析、报告生成 |
| 预览画布 | `apps/desktop/src/components/Export/ExportPreviewCanvas.tsx` | 366 | Canvas 实时帧渲染、进度条、缩略图网格 |
| 队列运行器 | `apps/desktop/src/export/export-queue-runner.ts` | 682 | 集成智能调度、可恢复导出、质量检查重试 |

### 测试覆盖

## 🙏 致谢

v4.27.0 开发过程中遵循了 Open Factory 的核心架构原则：
- ✅ 核心算法为纯函数，位于 `packages/editor-core`
- ✅ Timeline 突变通过 command 对象，不直接调用 Zustand setter
- ✅ 所有 Tauri 调用通过 `tauri-bridge.ts`
- ✅ FFmpeg 执行使用参数数组，不使用 shell 字符串
- ✅ 本地优先，无遥测、无登录、无云服务
