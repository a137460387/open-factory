## 🔑 关键技术决策

### 1. 格式去重策略
采用 `orientation:aspectRatio` 复合键对目标平台分组，相同方向和宽高比的平台共享裁剪方案，显著减少计算量。

### 2. 五维封面评分体系
- 清晰度（25%）+ 人脸（25%）+ 色彩（20%）+ 构图（15%）+ 运动模糊（15%）
- 支持人脸优先模式，确保封面主体突出

### 3. 平台规则表架构
每个平台独立定义节奏、字幕、片头片尾和优化策略，通过默认值回退机制处理未知平台，具备良好的可扩展性。

### 4. 渐进式 UI 集成
三标签页设计将平台选择、格式预览和适配建议分离，降低用户认知负担，同时支持一键智能推荐。

## 🚀 里程碑意义

**"多平台智能分发能力全面构建"** 是 Open Factory 发展历程中的重要里程碑：

1. **从编辑到分发**：项目从单一视频编辑工具扩展为支持多平台内容分发的创作中心
2. **智能适配**：7 大主流平台的深度适配，覆盖国内外主要视频平台
3. **自动化工作流**：从格式生成、封面制作到适配建议，实现端到端的自动化分发准备
4. **数据驱动**：基于五维评分的智能选帧和质量评估，确保分发内容质量
5. **用户友好**：一键推荐、批量分发、进度可视化，降低多平台运营门槛
