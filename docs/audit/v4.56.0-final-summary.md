## 关键技术决策

1. **Socket.IO + WebRTC**: 选择 Socket.IO 作为信令层，WebRTC 用于点对点数据传输，平衡了易用性和性能
2. **Redis 集群**: 房间状态持久化和跨实例同步，支持水平扩展
3. **沙箱隔离**: 插件在受限环境中运行，保护主进程安全
4. **链式 API**: SDK 采用流畅的链式设计，降低学习成本
5. **Docusaurus**: 选择成熟的文档框架，支持版本管理和搜索
