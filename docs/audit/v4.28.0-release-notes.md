# Open Factory v4.28.0 Release Notes

**发布日期：** 2026-07-16  
**Sprint：** C - 转场效果库与关键帧增强

---

## ✨ 26 种转场效果库

新增专业级转场效果系统，涵盖三大类别：

### 基础转场（10 种）
- **溶解 (Dissolve)**：经典交叉溶解效果
- **擦除 (Wipe)**：左、右、上、下四方向擦除
- **推移 (Push)**：相邻片段推移过渡
- **淡入淡出 (Fade)**：黑场/白场过渡
- **缩放 (Zoom)**：中心缩放过渡

### 进阶转场（10 种）
- **滑动 (Slide)**：多方向滑动效果
- **百叶窗 (Blinds)**：水平/垂直百叶窗
- **棋盘 (Checkerboard)**：棋盘格过渡
- **径向擦除 (Radial Wipe)**：圆形径向擦除
- **随机擦除 (Random Wipe)**：随机像素擦除

### 3D 转场（6 种）
- **立方体旋转 (Cube Rotate)**：3D 立方体翻转
- **门式过渡 (Door)**：开门/关门效果
- **折叠 (Fold)**：页面折叠过渡
- **虹膜 (Iris)**：虹膜收缩/扩张

### 技术特性
- FFmpeg `xfade` 滤镜完整导出支持
- 转场时长可自定义（默认 1 秒）
- 支持转场收藏与分类筛选
- 时间线拖拽直接应用转场

## 🎨 WebGL 实时预览面板

### GPU 加速渲染
- WebGL 2.0 shader 渲染管线
- 实时转场效果预览（悬停触发）
- 60fps 流畅预览体验

### 智能降级
- Canvas 2D 自动降级（WebGL 不可用时）
- 性能自适应质量调节
- 内存占用优化

### 交互特性
- 悬停预览：鼠标悬停转场卡片即时预览
- 拖拽应用：从预览面板拖拽到时间线片段
- 实时参数调整反馈

## 🎢 关键帧曲线编辑器增强

### 34 种缓动预设
- **基础缓动**：线性、 ease-in、ease-out、ease-in-out
- **二次/三次/四次**：Quadratic、Cubic、Quartic 曲线
- **指数/对数**：Exponential、Logarithmic 曲线
- **弹性/弹跳**：Elastic、Bounce、Back 效果
- **步进缓动**：Step Start、Step End、Step 中间值

### 贝塞尔手柄可视化
- 拖拽贝塞尔控制点实时调整曲线
- 手柄长度与角度可视化
- 曲线预览与数值显示同步

### 步进缓动支持
- 离散值跳变效果
- 可配置步数
- 适用于动画帧序列控制

## 🔗 迁移说明

### 新增模块路径
```typescript
// 转场效果库
import { TransitionLibrary, TransitionCategory } from '@open-factory/editor-core/src/transitions'

// 缓动预设
import { easingPresets, EasingPreset } from '@open-factory/editor-core/src/easing-presets'

// WebGL 预览组件
import { TransitionPreview } from '@/components/Transitions/TransitionPreview'
```

### 新增文件结构
```
packages/editor-core/src/
├── easing-presets.ts          # 34 种缓动预设定义
├── export/transitions/        # 转场导出逻辑
│   ├── transition-library.ts  # 转场效果注册表
│   └── xfade-builder.ts       # FFmpeg xfade 参数构建

apps/desktop/src/
├── components/Transitions/    # 转场 UI 组件
│   ├── TransitionLibrary.tsx  # 转场库面板
│   └── TransitionPreview.tsx  # WebGL 预览组件
└── store/transitionStore.ts   # 转场状态管理
```

### API 变更
- `EasingFunction` 类型扩展支持 `step-start`、`step-end`、`step(n)` 语法
- `TransitionConfig` 接口新增 `duration`、`easing` 属性
- `KeyframeEditor` 组件新增 `showBezierHandles` 属性
