# Open Factory v4.62.0 Release Notes

**Sprint AN: 安全漏洞清零与架构优化**

发布日期: 2026-07-22

---

## 🔒 安全修复

修复全部 **23 个安全漏洞**（1 Critical + 10 High + 12 Moderate/Low），实现漏洞清零。

### Critical
- **vitest** RCE 漏洞 — 升级至安全版本

### High
- **vite** 路径遍历漏洞 (CVE-2025-30208, CVE-2025-31125)
- **sharp** libvips 内存安全漏洞
- **Next.js** SSRF / DoS 漏洞 (CVE-2025-29927, CVE-2024-51479, CVE-2025-32419)
- **esbuild** 任意代码执行风险
- **nanoid** 可预测 ID 生成
- **cookie** 原型污染

### Moderate/Low
- **@babel/runtime** ReDoS
- **brace-expansion** ReDoS
- **path-to-regexp** ReDoS (CVE-2024-45296)
- **micromatch** ReDoS
- **ws** DoS (CVE-2024-37890)
- **http-proxy-middleware** 源代码泄露
- 及其他 6 项低危依赖漏洞

## 🧹 架构优化

### clamp 去重 Phase 1-2
- 将分散在各模块的 `clamp` 调用统一导入 `shared/math-utils`
- **119 → 15 处**内联 clamp，去重率 **87%**
- 新增 `packages/editor-core/src/utils/math.ts` 集中导出

### model-types.ts 类型解耦
- 将模型相关类型从巨型文件中独立提取
- 降低模块间耦合度

### exports 配置修复
- 修复 `@open-factory/editor-core` 缺失的 `./utils/time` 和 `./utils/math` 导出
- 修复 Vite 6 worker format 配置兼容性

## 🧪 测试加固

### Sync 模块覆盖率提升
- 新增 **59 个测试**覆盖多设备同步核心逻辑
- 覆盖率从 **69.93% → 88.92%**（+19%）

### 测试稳定性修复
- 修复 vitest 超时配置问题
- 修复 task-scheduler unhandled rejection 导致的测试崩溃
- 测试耗时从 **172.7s → 71.51s**（**-59%**）
