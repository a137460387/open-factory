# Open Factory v4.68.0 Release Notes

**Sprint AV — 自动化工作流**

> 从手动操作到自动化流水线，个人效率迎来终极进化。

---

## 核心特性

### ⚡ 宏录制与回放

录制剪辑操作序列，一键应用于其他片段，支持参数化调整。

- 录制器：实时捕获用户操作（裁剪、变速、调色、特效等 26 种操作类型）
- 回放引擎：按序执行宏操作，支持进度追踪和错误恢复
- 参数化宏：可定义可调参数（数值、颜色、选择等），应用时动态替换
- 持久化存储：宏文件支持导入/导出/分类管理，跨项目复用
- 录制面板：可视化控制录制状态、暂停/恢复、预览操作列表

### 🧩 节点式 AI 编排

可视化节点编辑器，拖拽串联 AI 引擎（高光检测 → 粗剪 → 配字幕），实现一键出片。

- 15+ 内置节点：视频输入、音频输入、AI 高光检测、AI 粗剪、字幕生成、音频分离、色彩校正、转场添加、导出等
- 连接验证：自动检测端口类型兼容性，防止无效连接
- 拓扑排序：自动计算节点执行顺序，支持并行分支
- 实时状态：节点运行状态可视化（idle/running/completed/error）
- 节点面板：拖拽式画布编辑器，缩放/平移/框选

### 📋 流程模板库

6 个预置工作流模板，开箱即用。

| 模板 | 说明 |
|------|------|
| 短视频一键出片 | 高光检测 → 粗剪 → 字幕 → 导出 |
| Vlog 自动处理 | 音频分离 → 降噪 → 调色 → 配乐 |
| 访谈精剪 | 语音识别 → 静音删除 → 字幕 → 导出 |
| 产品展示 | 素材排列 → 转场 → 文字叠加 → 导出 |
| 混剪模板 | 多素材 → 高光提取 → 节奏匹配 → 导出 |
| 播客处理 | 音频降噪 → 分段 → 转录 → 导出 |
