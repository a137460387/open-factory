## 🤖 自动化工作流引擎

可视化工作流编排引擎，支持触发器、条件判断和动作执行的完整自动化流水线。

### 核心功能
- **触发器系统** — 支持三种触发模式：导入触发（素材导入时自动执行）、手动触发（用户一键启动）、定时触发（基于时间规则自动执行）
- **条件判断引擎** — 灵活的条件表达式系统，支持文件类型、时长、分辨率、元数据等多维度条件匹配
- **动作执行器** — 丰富的预置动作：自动转码、元数据写入、标签分配、导出预设应用等
- **工作流链式执行** — 支持多步骤工作流串联，前一步骤输出作为后一步骤输入
- **执行历史与回溯** — 完整的工作流执行日志，支持查看每步执行结果和耗时

### 技术实现
- 1,069 行核心引擎代码，事件驱动架构
- 支持异步执行和取消操作
- 与时间线引擎通过命令对象集成
- 38 个单元测试覆盖所有引擎功能

## 🧠 智能场景分析

AI 驱动的素材智能分析系统，自动识别场景、生成标签、评估质量。

### 核心功能
- **自动场景检测** — 基于画面变化、音频特征和时间码自动识别场景切换点
- **内容标签生成** — 分析画面内容自动生成语义标签（人物、场景、动作、情绪等）
- **质量评估** — 多维度质量评分（清晰度、稳定性、曝光、色彩平衡）
- **场景聚类** — 相似场景自动归组，便于批量处理和素材管理
- **分析报告导出** — 生成结构化分析报告，支持 JSON/CSV 格式导出

### 技术实现
- 753 行分析引擎代码
- 可配置的分析精度和维度
- 支持增量分析，仅处理新增素材
- 25 个单元测试覆盖所有分析场景

## 📋 自动化规则系统

自定义规则引擎，将业务逻辑转化为可复用的自动化规则。

### 核心功能
- **规则定义** — 声明式规则语法，支持条件-动作模式
- **规则模板库** — 内置常用规则模板：自动归档、格式转换、元数据清理等
- **规则优先级** — 支持规则优先级排序和冲突检测
- **规则组合** — 多规则组合执行，支持与/或逻辑
- **规则导入导出** — 规则配置的持久化和分享

### 技术实现
- 857 行规则引擎代码
- 基于表达式树的条件求值
- 规则热加载，无需重启应用
- 32 个单元测试覆盖所有规则功能

## 🎨 可视化编辑面板

直观的工作流和分析结果可视化界面。

### 核心功能
- **工作流编辑器** — 拖拽式工作流编排，可视化节点连接
- **素材分析视图** — 分析结果的图形化展示，支持时间线叠加
- **执行状态面板** — 实时显示工作流执行进度和状态
- **规则管理界面** — 规则的创建、编辑、启用/禁用管理

### 技术实现
- AutomationPanel（479 行）— 工作流编辑与管理主面板
- SceneAnalysisView（539 行）— 场景分析结果可视化
- useAutomationWorker Hook（264 行）— Worker 通信桥接
- automation.worker.ts（244 行）— 后台分析 Worker
