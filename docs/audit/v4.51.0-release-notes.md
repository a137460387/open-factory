## 核心功能

### AI 智能质检引擎

多维度质量检测引擎，为视频/音频内容提供专业级质检能力。

- **黑场检测** — 基于像素亮度分析，自动识别黑帧、彩条测试图案、静帧等技术缺陷
- **爆音检测** — RMS/峰值分析，检测音频削波、静音段、失真等音频问题
- **节奏分析** — 场景切换检测 + CPM（每分钟切换数）曲线，识别过慢/过快节奏段落
- **平台合规检查** — 支持 YouTube 1080p/4K、TikTok 9:16、Instagram Reel、Twitter、Broadcast PAL/NTSC 等 8 种平台规范
- **可视化质检报告** — 综合评分 (A-F 等级)、问题分类统计、可自动修复标记、时间轴定位

### 本地资源智能管家

本地资源智能管理系统，释放磁盘空间、优化工作流效率。

- **代理文件生成** — 大文件自动识别，生成低分辨率代理用于流畅编辑（640x360, 1Mbps）
- **智能缓存清理** — 过期缓存检测、分类统计（预览/渲染/AI分析/缩略图/波形）、一键清理
- **素材去重** — 基于哈希的精确去重 + 感知哈希相似度检测（95%+ 阈值），避免存储浪费
- **未使用文件识别** — 基于最后访问时间，识别 30 天以上未使用的素材文件
- **清理建议系统** — 风险评估（低/中/高）、自动清理标记、可释放空间统计

### 性能监控中心

实时性能监控与优化建议中心。

- **CPU/GPU/内存实时监控** — 1 秒采样间隔，系统资源使用率可视化
- **瓶颈检测** — 自动识别 CPU/内存/GPU/磁盘/IO/算法瓶颈，严重程度分级
- **优化建议** — 硬件/软件/工作流/设置四维优化建议，预估改善效果
- **趋势分析** — 历史数据趋势追踪，识别性能退化
- **任务性能追踪** — 渲染/导出/AI处理等任务的资源消耗分析

## 技术实现

- 质检引擎: 纯本地算法实现，无外部依赖，支持 Web Worker 并行处理
- 资源管理: 异步 I/O + 后台处理，不阻塞主线程
- 性能监控: requestAnimationFrame + Performance API，低开销采样
- 所有模块遵循不可变数据模式，纯函数设计，易于测试和扩展

## 文件变更

- 新增模块: `quality/` (质检引擎), `resources/` (资源管家), `performance/` (性能监控)
