# Open Factory v4.33.0 Release Notes

**发布日期：** 2026-07-17  
**Sprint：** H — 多机位编辑基础  
**里程碑：** 四大专业支柱（视频、音频、字幕、多机位）全面成型

---

## 🎥 多机位同步

### 基于音频波形的自动对齐
- 新增 `AudioSyncAnalyzer` 模块，通过音频波形特征提取实现多机位素材的自动时间对齐
- 支持跨机位音频指纹匹配，自动检测同步偏移量
- 提供同步置信度评分，低于阈值时提示手动校验

### 时间码手动同步
- 支持基于时间码（Timecode）的手动同步方式
- 可视化同步偏移指示器，精确调整各机位时间线对齐
- 同步状态实时监控（`SyncStatusMonitor`），对齐状态一目了然

---

## 🖥️ 高性能预览

### Web Worker + OffscreenCanvas 并行解码
- 新增 `multicam-preview.worker.ts`，将多机位预览解码任务卸载至 Web Worker
- 使用 OffscreenCanvas 实现真正的并行渲染，主线程零阻塞
- 支持最多 9 路机位同时预览解码

### HybridPreviewGrid 混合渲染回退
- `HybridPreviewGrid` 组件自动检测浏览器 OffscreenCanvas 支持
- 不支持时自动降级至主线程 Canvas 渲染，保证兼容性
- 渲染性能自适应：根据机位数量动态调整预览分辨率

---

## ⌨️ 实时切换

### 键盘快捷键实时切换机位
- 数字键 `1`-`9` 实时切换当前活跃机位
- 切换操作零延迟，预览画面即时响应
- 支持切换时自动在时间线上生成剪辑片段

### 自动生成剪辑片段
- `SwitchGenerator` 根据切换操作自动生成多机位剪辑片段
- 片段包含机位来源、切换时间点、持续时长等元数据
- 支持后续手动微调切换点位置

---

## 🎚️ 多机位混音

### 机位音频分组管理
- `AudioGroupingManager` 支持将各机位音频按组管理
- 可设定主音频源和备用音频源
- 支持音频跟随视频切换自动切换

### 独立混音模式
- 支持"跟随切换"和"独立混音"两种模式
- 跟随模式：音频随视频机位切换自动切换
- 独立模式：各机位音频独立控制，手动调节混音比例

## 🏗️ 架构亮点

### 命令模式时间线操作
- 所有多机位切换操作通过命令对象执行，符合项目架构规范
- 支持撤销/重做，保证操作可追溯

### Web Worker 隔离
- 多机位预览解码完全在 Worker 中执行
- 主线程仅负责 UI 渲染和用户交互
- 通过 `postMessage` 进行 Worker 通信，数据序列化高效

### 本地优先原则
- 所有媒体处理保持本地执行，无云端依赖
- 预览使用本地代理媒体，导出使用原始源文件
- 符合 open-factory 本地优先的核心设计理念
