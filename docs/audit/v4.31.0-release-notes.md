# Open Factory v4.31.0 Release Notes

**发布日期**: 2026-07-17  
**Sprint**: F - 字幕工作流改进  
**里程碑**: 音频-字幕-视频完整后期制作工作流形成

---

## 📝 可视化字幕编辑器

全新的可视化字幕编辑面板，采用四标签页设计：

- **列表标签页**: 字幕条目列表展示，支持拖拽排序、时间轴对齐
- **查找替换标签页**: 全文搜索与批量替换，支持正则表达式
- **样式标签页**: 单条/批量字幕样式编辑（字体、颜色、阴影、描边等）
- **批量操作标签页**: 批量时间偏移、批量删除、批量样式应用

**核心特性**:
- 实时 Canvas 预览，所见即所得
- 与时间轴深度集成，双向同步
- 支持撤销/重做操作历史

---

## 🎨 样式预设系统

完整的字幕样式预设管理：

- **预设 CRUD**: 创建、读取、更新、删除自定义预设
- **导入导出**: JSON 格式预设文件，支持跨项目复用
- **收藏搜索**: 预设收藏夹，关键词搜索过滤
- **内置预设**: 包含标准、电影、社交媒体等多种预设模板

**模块路径**: `@open-factory/editor-core/subtitles/style-presets`

---

## 🌍 多语言字幕导出

支持多语言字幕的统一管理与导出：

- **导出格式**:
  - SRT (SubRip) - 最通用格式
  - VTT (WebVTT) - Web 标准格式
  - ASS (Advanced SubStation Alpha) - 高级样式格式
- **FFmpeg 嵌入**: 支持将字幕硬编码到视频流中
- **语言分组**: 按语言代码分组管理，批量导出

**模块路径**: `@open-factory/editor-core/subtitles/multi-language-export`

---

## ⚡ Canvas 60fps 渲染优化

字幕 Canvas 渲染器性能大幅优化：

- **脏检查优化**: 仅重绘变化区域，减少不必要的全量渲染
- **OffscreenCanvas**: 利用离屏 Canvas 进行预渲染
- **区域清除**: 精确清除字幕区域，避免全画布清除开销
- **60fps 目标**: 在主流硬件上稳定达到 60fps 渲染

**模块路径**: `@open-factory/editor-core/subtitles/canvas-renderer`

## 🎯 里程碑意义

**v4.31.0 标志着 Open Factory 形成了"音频-字幕-视频"完整的后期制作工作流**：

- **音频处理** (v4.30.0): 多轨混音器、智能降噪、频谱分析
- **字幕工作流** (v4.31.0): 可视化编辑、样式预设、多语言导出
- **视频编辑** (持续迭代): 时间轴、转场、特效、导出

至此，Open Factory 已具备专业级视频后期制作的完整能力链。
