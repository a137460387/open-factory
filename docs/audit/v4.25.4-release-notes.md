# Open Factory v4.25.4 — 安全审计与工程质量提升

## 🔒 安全修复

本版本基于 2026-07-15 全量代码审计（100 个问题），完成了所有 Critical 和 High 级别问题的修复。

### Critical 修复

| 问题 | 修复内容 |
|------|----------|
| **WebDAV 密码加密密钥可预测** | 密码存储从可预测的 SHA-256 派生迁移到系统 keyring，使用操作系统原生安全存储 |
| **中文分词完全失效** | 实现 n-gram 分词策略（bigram + unigram 混合），中文关键词提取、话题聚类功能恢复正常 |

### High 安全修复

| 问题 | 修复内容 |
|------|----------|
| WebDAV nonce 可预测 | 随密码存储一起迁移到 keyring，不再需要 nonce |
| Asset Protocol scope 过宽 | 从 8 个目录收窄至 `$APPDATA`/`$APPCACHE`/`$TEMP` |
| CSP connect-src 缺少域名 | 补充 `gist.githubusercontent.com` |

### High 业务逻辑修复

| 问题 | 修复内容 |
|------|----------|
| TF-IDF 评分公式错误 | 修正为 `frequency * Math.log(1 + totalTokens / (frequency + 1))` |
| 导出取消后无资源清理 | 添加 finally 块清理临时文件和子进程 |
| 导出格式校验缺失 | 添加 `SUPPORTED_FORMATS` 白名单校验 |
| 轨道锁定未被命令检查 | 为 6 个关键时间线命令添加 `assertClipsNotOnLockedTrack` 检查 |
| DeleteClipsCommand 未清理 Transition | 删除片段时同步清理关联的 Transition |
