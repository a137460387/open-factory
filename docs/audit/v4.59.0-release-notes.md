# Open Factory v4.59.0 Release Notes

**发布日期**: 2026-07-22
**Sprint**: AI — 全量审计与性能优化

---

## 🔒 安全修复

- **移除 drizzle-orm**：消除 HIGH 级 SQL 注入漏洞风险（该依赖未被使用，属于冗余攻击面）
- **升级 dompurify 至 3.4.12**：修复已知安全漏洞，增强 HTML 净化能力

## 🧹 依赖精简

- **移除 8 个未使用依赖**：减少供应链攻击面和维护负担
- **api-gateway 依赖从 13 项精简至 7 项（-46%）**：仅保留实际使用的依赖
- **移除冗余 package-lock.json**：统一使用 bun.lock 管理依赖锁定

## 🧪 测试覆盖提升

- **新增 168 个测试**：
  - SDK 模块：62 个测试
  - api-client 模块：21 个测试
  - quality-panel：49 个测试
  - resource-panel：36 个测试
- **quality-panel.ts 覆盖率：0% → 100%**
- **resource-panel.ts 覆盖率：0% → 100%**
- **测试总数：7926 个（472 个测试文件）**，全部通过

## 📐 架构审计

- **循环依赖分析**：确认 18 个循环依赖均为 `import type`（TypeScript 类型导入），运行时无害，不影响打包和运行
- **SDK 与 editor-core 类型独立性确认**：两个包的类型系统互不干扰
- **架构文档更新**：`docs/architecture.md` 已更新，反映当前架构状态
