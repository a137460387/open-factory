# Open Factory v4.65.0 Release Notes

**发布日期**: 2026-07-23  
**Sprint**: AR - 沉浸式创作体验

---

## 核心特性

### ⚡ WebGPU 渲染引擎

全面接管视频解码与效果渲染，4K 实时回放零延迟。

- **GPU 硬件加速解码**: 利用 WebGPU Compute Shader 实现视频帧解码，支持 H.264/H.265/VP9/AV1
- **并行效果渲染**: 特效、转场、调色全部在 GPU 并行执行，告别 CPU 瓶颈
- **智能内存管理**: 自动管理 GPU 显存，支持 4K/8K 素材流畅编辑
- **实时预览**: 效果参数调整即时反馈，所见即所得

**性能提升**: 4K 视频回放从 15fps → 60fps，效果渲染速度提升 8-12 倍

---

### 🧠 智能代理系统

基于设备性能自动生成代理，老旧设备流畅剪辑 4K。

- **自动性能检测**: 启动时检测 CPU/GPU/内存，生成设备性能画像
- **智能代理规划**: 根据素材分辨率和设备能力，自动决定是否需要代理
- **后台生成**: 代理文件在后台异步生成，不阻塞编辑流程
- **无缝切换**: 编辑时使用代理，导出时自动切换原始素材

**兼容性**: 低配设备（集成显卡/8GB 内存）也能流畅编辑 4K 素材

---

### 🚀 增量渲染引擎

仅渲染修改部分，导出效率提升数十倍。

- **修改检测**: 精确追踪时间线上每个片段的修改状态
- **智能缓存**: 已渲染片段自动缓存，避免重复计算
- **增量导出**: 仅重新渲染修改过的片段，其余直接复用缓存
- **断点续渲**: 导出中断后可从断点继续，无需重新开始

**效率提升**: 多次修改后导出，从 10 分钟 → 30 秒

---

### 🧘 Zen 专注模式

一键隐藏非必要 UI，沉浸式心流创作。

- **一键切换**: 快捷键或按钮一键进入专注模式
- **智能隐藏**: 自动隐藏工具栏、面板、菜单，只保留预览窗口
- **自定义布局**: 可配置哪些 UI 元素在专注模式下保留
- **动画过渡**: 平滑的 UI 淡入淡出，不打断创作节奏

**适用场景**: 精细调色、音频调整、创意探索时的沉浸式工作

---

### ⌨️ 全局快捷键体系

支持自定义映射与可视化编辑，操作效率倍增。

- **可视化编辑器**: 直观的快捷键配置界面，支持搜索和冲突检测
- **自定义映射**: 所有功能均可自定义快捷键，支持组合键
- **预设方案**: 内置 Premiere/FCP/DaVinci 风格快捷键方案
- **快捷键提示**: 悬停按钮时显示对应快捷键，加速学习

**效率提升**: 高频操作速度提升 40-60%

---

### 🎨 个性化主题引擎

界面颜色/布局深度定制，打造专属创作空间。

- **颜色定制**: 主色调、强调色、背景色全面可调
- **布局保存**: 保存和切换多套界面布局方案
- **主题导入导出**: 分享和导入社区主题
- **深色/浅色模式**: 支持系统主题跟随和手动切换

**自定义程度**: 15+ 可调颜色参数，5+ 布局预设

## 升级指南

### 从 v4.64.0 升级

1. **无需迁移**: 所有新功能向后兼容
2. **首次启动**: 智能代理系统会自动检测设备性能
3. **快捷键**: 首次启动会提示选择快捷键方案
4. **主题**: 默认使用系统主题，可在设置中自定义

### 推荐配置

- **最低配置**: 集成显卡 + 8GB 内存（自动启用代理模式）
- **推荐配置**: 独立显卡（GTX 1060+）+ 16GB 内存
- **最佳体验**: RTX 3060+ / RX 6600+ + 32GB 内存
