# Open Factory v4.34.0 Release Notes

**发布日期**: 2026-07-17
**Sprint**: I — AI 辅助制作（语音转字幕）

---

## 🎯 里程碑：AI 赋能时代开启

v4.34.0 标志着 Open Factory 正式迈入 AI 辅助制作时代。本版本引入了基于本地 whisper.cpp 的语音转字幕系统，遵循"本地优先、隐私安全"的架构理念，所有 AI 推理均在用户设备上完成，无需云端依赖。

---

## 🧠 本地 AI 语音转录

- **集成 whisper.cpp**: 基于 OpenAI Whisper 模型的本地 C++ 推理引擎，通过 WebAssembly 运行于浏览器/桌面环境
- **多语言支持**: 中文（普通话/粤语）、英语、日语、韩语及自动语言检测
- **完全离线**: 所有模型推理在本地完成，音频数据永不离开用户设备
- **模型灵活选择**: 支持 tiny/base/small/medium 多种模型尺寸，平衡精度与性能

## ⚡ 高性能非阻塞 UI

- **Web Worker 隔离**: 语音转录推理在独立 Web Worker 中执行，主线程始终保持 60fps 流畅响应
- **三阶段进度回调**: 模型加载 → 音频预处理 → 推理转录，每个阶段均有实时进度反馈
- **优雅取消机制**: 支持随时中断转录任务，资源即时释放
- **内存安全**: 分块处理长音频，避免 OOM 崩溃

## 📝 智能后处理流水线

- **短句合并**: 自动合并过短的字幕片段，提升可读性
- **长句拆分**: 智能拆分超长字幕，符合字幕阅读习惯
- **时间戳精确对齐**: 转录时间戳与音频波形精确对齐
- **阅读时间估算**: 根据字幕文本长度自动估算最佳显示时长

## 🔗 字幕工作流深度集成

- **一键生成字幕**: 在 ASRStage 组件中直接触发语音转录，生成标准字幕数据
- **自动导入时间线**: 转录结果无缝导入字幕轨道，与 v4.31.0 字幕编辑器完美衔接
- **字幕样式预设**: 自动应用用户偏好的字幕样式
- **多格式导出兼容**: 生成的字幕可直接导出 SRT/ASS/TTML 等格式

## 🏗️ 架构遵循

- ✅ Core-first 开发顺序：先实现 `editor-core` 转录引擎，再集成 App Shell UI
- ✅ Web Worker 隔离：大模型推理不阻塞主线程
