## 🏗️ 架构遵循情况

### ✅ 遵循的架构原则

1. **命令模式**：时间线操作通过 command 对象执行，未直接调用 Zustand setter
2. **模块化设计**：转场效果独立为 `@open-factory/editor-core/src/export/transitions` 模块
3. **测试覆盖**：新增模块均有完整单元测试覆盖
4. **类型安全**：所有新增代码通过 TypeScript 严格类型检查
5. **本地优先**：无云端依赖，WebGL 渲染完全本地执行
