**发布日期**: 2026-07-23
**Sprint**: AQ - 个人创作者赋能

---

## 核心功能

### ✂️ 智能粗剪
基于音频分析的口误/卡顿检测，一键生成精剪时间线。自动识别并标记口语中的停顿、重复和口误片段，大幅减少手动剪辑时间。

**关键文件**: `packages/editor-core/src/smart-stutter-detection.ts`

### 🏷️ AI 素材打标
本地 AI 自动识别场景内容并生成标签。无需上传到云端，所有处理在本地完成，保护创作者隐私。

**关键文件**: `packages/editor-core/src/ai-scene-tagger.ts`

### 📱 社交媒体预设
内置 15 个主流平台导出预设，包括：
- B站、YouTube、抖音、快手
- Instagram、TikTok、Twitter/X
- 微信视频号、小红书、西瓜视频
- Facebook、Vimeo、LinkedIn、Pinterest、Snapchat

每个预设自动配置分辨率、码率、帧率和编码格式。

**关键文件**: `packages/editor-core/src/export/social-media-presets.ts`

### 🚀 批量渲染增强
支持后台连编队列及渲染后自动关机/休眠。适合长时间批量渲染场景，渲染完成后自动执行系统操作。

**关键文件**: `packages/editor-core/src/export/batch-render-actions.ts`

### ☁️ 网盘直连
支持直接挂载 OneDrive/WebDAV 网盘作为媒体库。无需同步到本地，直接从云端读取和编辑素材。

**关键文件**: `packages/editor-core/src/personal-cloud.ts`

## 技术实现

- 音频分析使用 Web Audio API 进行实时波形检测
- AI 打标基于本地 ONNX Runtime 推理，无需网络连接
- 社交预设采用策略模式，易于扩展新平台
- 批量渲染使用 Promise 队列管理并发
- 网盘直连通过 WebDAV 协议和 Microsoft Graph API 实现
