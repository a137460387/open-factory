## 🎨 AI色彩分级

全新的AI驱动色彩分级系统，让专业调色变得简单高效。

### 核心功能

| 功能 | 说明 |
|------|------|
| **智能色彩匹配** | `matchColors()` — 不同镜头间自动色彩风格匹配，支持直方图匹配算法 |
| **AI自动分级** | `autoGradeImage()` — 基于场景内容自动调整色彩，支持10种风格、8种场景类型 |
| **风格迁移** | `transferStyle()` — 参考图像风格应用到目标画面，支持色彩映射和直方图匹配 |
| **色彩校正** | `applyColorCorrection()` — 完整色彩校正工具链：亮度/对比度/饱和度/色温/色调/伽马/Lift/Gamma/Gain |
| **色彩分析** | `analyzeImageColors()` — 全面分析亮度/对比度/饱和度/色温/色调/主色调/直方图/动态范围 |

### 支持的风格预设
- 电影级（Cinematic）、暖色调（Warm）、冷色调（Cool）
- 复古（Vintage）、黑白（B&W）、高对比度（High Contrast）
- 柔和（Soft）、鲜艳（Vivid）、戏剧性（Dramatic）、自然（Natural）

### 支持的场景类型
- 室内/室外、日景/夜景、人像/风景/动作/特写

## 🧠 场景理解增强

基于计算机视觉的场景智能分析系统。

### 核心功能

| 功能 | 说明 |
|------|------|
| **物体检测与跟踪** | `detectObjects()` + `trackObjects()` — 基于颜色分割和轮廓检测，支持11种物体类别，IoU匹配跨帧跟踪 |
| **人脸检测与表情分析** | `detectFaces()` — 基于肤色检测和椭圆拟合，含关键点/表情/头部姿态分析 |
| **动作识别** | `recognizeActions()` — 基于帧间运动分析，支持运动分类和强度评估 |
| **语义分割** | `segmentSemantics()` — 基于位置和颜色的场景分割，8种语义类别（天空/地面/建筑/植被等） |
| **场景描述** | `describeScene()` — 综合场景分析，含类型/氛围/光照/时间段自动生成 |

### 技术特点
- 纯本地计算，无需云端API
- 支持逐帧分析和批量处理
- 完整的置信度评分系统

## ✂️ 智能剪辑增强

音频驱动的智能剪辑系统，让剪辑更高效。

### 核心功能

| 功能 | 说明 |
|------|------|
| **音频节拍检测** | `detectBeats()` — 音频能量包络分析，自动检测BPM和节拍点 |
| **情绪感知剪辑** | `analyzeEmotion()` + `emotionAwareEdit()` — 音频特征推断情绪，根据情绪变化建议剪辑点 |
| **自动预告片生成** | `generateTrailer()` — 智能生成预告片，支持5种风格、4种节奏模式 |
| **智能片段排序** | `sortSegments()` — 6种排序策略（叙事/情绪/节奏/重要性/时长/随机） |
| **节奏匹配** | `rhythmMatchEdit()` — 根据音频节拍自动对齐剪辑点 |
| **剪辑建议** | `generateCutSuggestions()` — 节拍/情绪/场景变化综合剪辑点建议 |

### 预告片风格
- 动作（Action）、剧情（Drama）、喜剧（Comedy）、悬疑（Mystery）、纪录片（Documentary）

### 节奏模式
- 快节奏（Fast）、中等（Medium）、慢节奏（Slow）、动态（Dynamic）

## 🎨 ACES色彩管理

完整的ACES（Academy Color Encoding System）色彩管理实现。

### 色彩空间转换

| 转换路径 | 说明 |
|----------|------|
| sRGB ↔ Rec.709 | 标准高清色彩空间互转 |
| Rec.709 ↔ Rec.2020 | 高清到超高清色彩空间扩展 |
| ACES2065-1 (AP0) | ACES归档色彩空间，全色域线性编码 |
| ACEScg (AP1) | ACES计算色彩空间，适合合成和渲染 |
| ACEScct | ACES对数编码色彩空间，适合调色 |

### 传输特性

| 特性 | 说明 |
|------|------|
| sRGB | 标准RGB传输特性线性化 |
| Gamma 2.2 | 传统Gamma编码 |
| PQ (Perceptual Quantizer) | HDR感知量化，支持HDR10/Dolby Vision |
| HLG (Hybrid Log-Gamma) | 广播HDR传输特性 |
| ACEScct | ACES对数传输特性 |

### 色调映射算法

| 算法 | 说明 |
|------|------|
| **Reinhard** | 经典全局色调映射 |
| **ACES Hill** | ACES标准Hill色调映射 |
| **ACES Narkowicz** | ACES Narkowicz快速近似 |
| **Filmic** | 电影风格色调映射 |
| **AGX** | 现代高动态范围色调映射 |

### LUT管理

- **1D LUT** — 单通道查找表，用于简单色彩变换
- **3D LUT** — 三通道立方体查找表，用于复杂色彩变换
- **三线性插值** — 3D LUT高质量插值算法
- **.cube文件解析** — 支持标准.cube格式LUT文件导入
- **OCIO配置** — 自动生成OpenColorIO配置文件

## 🏗️ 架构设计原则

1. **本地优先** — 所有AI推理在本地完成，不依赖云端API
2. **纯计算函数** — 无副作用，易于测试和复用
3. **TypeScript类型安全** — 完整的接口定义和类型约束
4. **配置驱动** — 所有模块支持可配置参数和合理默认值
5. **验证/归一化** — 每个参数类型提供 `createDefault`/`validate`/`normalize` 函数三件套
6. **ACES标准** — 遵循ACES色彩管理标准，确保色彩准确性
