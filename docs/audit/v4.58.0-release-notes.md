## 🔌 前后端数据打通

### Fastify API 网关

新增 `packages/api-gateway`，基于 Fastify 构建高性能 REST API 服务：

- **10 个 RESTful 端点**：插件搜索/详情/安装/评论、创作者仪表板/收入/统计/资料更新
- **JWT 认证**：HS256 签名，支持 access token + refresh token 双令牌机制
- **RBAC 权限**：admin/creator/user 三级角色，细粒度资源-操作权限控制
- **Zod 校验**：请求参数自动 schema 验证，类型安全的错误响应
- **Swagger 文档**：自动生成 OpenAPI 规范文档

### 类型安全的 HTTP 客户端

新增 `packages/api-client`，提供类型安全的 API 调用层：

- **`OpenFactoryApiClient` 类**：封装所有 API 端点，自动处理认证头和错误
- **React Hooks**：`usePluginSearch`、`usePluginDetail`、`useCreatorDashboard`、`useCreatorProfile`、`useCreatorStats`、`useCreatorRevenue`
- **单例模式**：`initializeApiClient()` + `getApiClient()` 全局初始化
- **三端类型对齐**：api-client ↔ api-gateway ↔ plugin-market 类型完全一致

### 前端数据接入

- **创作者后台**：`useCreator`、`useAnalytics`、`useRevenue` hooks 已接入 api-client，前端从 API 获取真实数据结构
- **插件市场**：类型转换层就绪，可无缝切换到 api-gateway 后端

## 🤖 本地生成式 AI

### Wan2.1 模型集成

新增 `packages/ai-generator`，集成阿里万象 Wan2.1-I2V-14B-720P-Turbo 模型：

- **文本生成视频**：text prompt → 文本编码 → 帧序列生成 → 视频编码
- **图片生成视频**：输入图片 → 图片编码 → 运动估计 → 帧序列生成 → 视频编码
- **运动强度控制**：可调 motion strength 参数控制生成视频的动态程度
- **分辨率支持**：720P (1280×720) 和 480P (854×480)
- **帧率可配**：默认 30fps，支持 16/24/30 fps

### WebGPU 加速与自动降级

- **WebGPU 后端**：GPU compute shader 加速推理，最大 256MB 缓冲区
- **WebGL 后端**：WebGL2 fallback，128MB 缓冲区
- **CPU 后端**：纯 CPU 推理兜底，512MB 缓冲区
- **自动降级链**：WebGPU → WebGL → CPU 无缝切换，无需用户干预
- **能力检测**：运行时自动检测 GPU 能力、最大纹理尺寸、F16/INT8 支持

### 模型管理

- **模型清单**：支持 Wan2.1 多个变体（14B/1.3B，I2V/T2V）
- **LRU 缓存**：智能模型缓存，自动淘汰最久未用模型
- **量化支持**：FP32 → FP16 → INT8 → INT4 四级精度，默认 INT8
- **校验和验证**：SHA-256 模型文件完整性校验
- **配置系统**：Zod schema 验证，环境变量覆盖，单例模式

### 支持的模型

| 模型 | 参数量 | 用途 | 默认精度 |
|------|--------|------|----------|
| Wan2.1-I2V-14B-720P-Turbo | 14B | 图片转视频 | INT8 |
| Wan2.1-I2V-1.3B-480P | 1.3B | 轻量图片转视频 | INT8 |
| Wan2.1-T2V-14B-720P | 14B | 文本转视频 | INT8 |

## 🏗️ 架构决策

1. **Fastify over Express** — 更好的 TypeScript 支持、schema 验证、性能
2. **单例 API 客户端** — `initializeApiClient()` + `getApiClient()` 模式，React 组件共享同一实例
3. **结构化类型对齐** — api-client 类型与 plugin-market 类型完全一致，避免类型转换
4. **WebGPU 自动降级** — 运行时检测 + 无缝切换，最大化兼容性
5. **INT8 默认精度** — 14B 模型最佳质量/体积平衡点
6. **Zod 配置校验** — 类型安全的配置管理，支持环境变量覆盖
