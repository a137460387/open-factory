## 🎉 版本亮点

v4.45.0 完成了**多平台智能分发能力的全面构建**，新增多格式生成引擎、平台适配系统和智能封面生成三大核心模块，并通过完整的 UI 集成提供一站式多平台分发体验。

---

## 📱 多格式生成引擎

全新多格式生成引擎，从单一源视频自动派生多种画面格式：

- **横版 / 竖版 / 方形自动生成**：根据目标平台需求，自动裁剪生成 16:9、9:16、1:1 等多种画面比例
- **智能裁剪主体居中**：基于人脸检测和主体识别，确保裁剪后主体始终位于画面中心
- **格式去重优化**：相同方向和宽高比的平台共享裁剪方案，避免冗余计算
- **质量评估系统**：对每个裁剪结果进行质量评分，基于宽高比差异、裁剪面积和方向变化三维度评估
- **预览数据生成**：自动计算裁剪区域在预览容器中的位置和 FFmpeg 滤镜链

## 🎯 平台适配系统

针对主流视频平台的深度适配与优化：

- **YouTube 1080p**：中等节奏，8 片段/分钟，平滑转场 0.5s，含标题卡 intro 和片尾 outro
- **YouTube Shorts**：快节奏，15 片段/分钟，硬切转场 0.2s，循环友好
- **抖音/TikTok**：快节奏，18 片段/分钟，炫酷转场 0.15s，3s 开场 hook，循环友好
- **B站**：中等节奏，10 片段/分钟，平滑转场 0.4s，说话人标签，半透明字幕背景
- **小红书**：中等节奏，12 片段/分钟，平滑转场 0.3s，胶囊字幕背景，主体上移
- **微信视频号**：慢节奏，6 片段/分钟，平滑转场 0.6s，生活方式内容
- **快手**：快节奏，15 片段/分钟，硬切转场 0.2s，2s hook，循环友好

每个平台包含独立的节奏配置、字幕样式、片头片尾和优化策略。适配分析功能可自动检测项目设置与目标平台的差异并生成优化建议。

## 🖼️ 智能封面生成

基于五维评分体系的智能封面生成：

- **清晰度评分**（25%权重）：基于 Laplacian 方差评估图像锐度
- **人脸检测**（25%权重）：优先选择包含清晰人脸的帧
- **色彩丰富度**（20%权重）：评估色彩饱和度和多样性
- **构图评分**（15%权重）：基于三分法则评估画面构图
- **运动模糊检测**（15%权重）：排除运动模糊严重的帧

封面生成支持：
- 5 种内置封面尺寸预设（YouTube 1280×720、竖版 1080×1920、方形 1080×1080、Pinterest 1000×1500、小红书 1080×1440）
- 品牌水印叠加（Logo、文字或组合），支持位置、缩放和透明度配置
- 平台专属封面样式和渐变遮罩

## 🎨 UI 集成

全新三标签页智能分发面板：

- **平台选择标签页**：平台卡片展示（分辨率、宽高比、帧率、码率、最大时长），推荐评分，适配标签（节奏风格、hook 时长、循环优化）
- **格式预览标签页**：裁剪区域可视化，质量指示条，目标平台标签
- **适配建议标签页**：按严重程度分级的优化建议（关键/警告/信息），颜色编码

操作栏支持：
- 一键智能推荐（自动选择评分 > 0.4 的平台）
- 全选 / 清除平台
- 批量导出进度展示，逐平台进度条
- 一键分发到 N 个平台
