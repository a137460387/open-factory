# Open Factory v4.41.0 Release Notes

**发布日期**: 2026-07-18
**Sprint**: P — 团队管理与多设备同步
**里程碑**: 团队协作能力全面提升

---

## 🤝 团队管理

完整的团队协作基础设施，支持多人协同编辑项目。

### 核心功能
- **团队创建与管理** — 创建团队、设置名称/描述/头像、管理团队配置
- **成员管理** — 添加/移除成员、角色变更、成员状态管理（活跃/非活跃/已离开）
- **角色权限系统** — 所有者/管理员/编辑者/查看者四级角色体系
- **邀请系统** — 发送邀请、接受/拒绝邀请、邀请过期管理
- **项目共享** — 团队内项目共享、共享权限控制
- **审计日志** — 完整的操作审计记录，支持按时间/操作类型/成员筛选
- **状态快照** — 团队状态导出/导入，支持团队配置备份与恢复

### 技术实现
- 事件驱动架构，支持实时状态变更通知
- 模块化设计，核心逻辑与 UI 解耦
- 44 个单元测试覆盖所有核心功能

---

## 📱 多设备同步

跨设备无缝同步，支持离线工作与冲突自动解决。

### 核心功能
- **设备注册与管理** — 设备注册、状态追踪、设备信息管理
- **变更集同步** — 基于变更集的差量同步，高效传输
- **冲突检测与解决** — 自动冲突检测、自动/手动解决策略
- **离线队列** — 离线操作自动排队，恢复连接后自动重试
- **WebSocket 传输** — 实时双向通信，适配器模式支持灵活扩展
- **数据压缩** — 传输数据压缩，减少带宽占用
- **校验和验证** — 数据完整性校验，确保同步准确性

### 技术实现
- WebSocket 适配器模式，支持 Mock 测试
- 离线队列带重试策略与指数退避
- 差量同步减少数据传输量
- 11 个单元测试覆盖核心同步逻辑

---

## 🔒 高级权限系统

细粒度权限控制，支持项目/文件夹/文件级别的权限管理。

### 核心功能
- **多级权限控制** — 项目级、文件夹级、文件级权限设置
- **权限继承** — 三种继承模式：严格模式、宽松模式、覆盖模式
- **临时权限** — 带过期时间的临时权限，自动撤销
- **权限组管理** — 创建权限组、批量权限分配
- **权限评估** — 权限计算引擎，支持缓存加速
- **审计日志** — 完整的权限变更审计记录

### 技术实现
- 权限评估缓存，优化高频权限查询
- 灵活的继承策略，适应不同团队工作流
- 46 个单元测试覆盖所有权限场景
