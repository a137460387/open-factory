## 技术亮点

1. **风格指纹 → 模板参数映射**：复用 `style-analyzer` 的多维风格数据（节奏、色彩、转场、音频、效果），自动推断模板类别并生成结构化模板参数。

2. **.oft 文件格式**：JSON-based 模板文件，SHA-256 校验和确保完整性，支持版本兼容性检查和渐进式迁移。

3. **CRDT 共享类型设计**：将 Timeline/Track/Clip/Transition 映射为 Yjs 兼容的共享类型，支持属性级细粒度合并和 Lamport 时间戳冲突解决。

4. **E2E 加密方案**：ECDH 密钥交换 + AES-GCM-256 内容加密，使用 Web Crypto API 实现，无需第三方加密库。
